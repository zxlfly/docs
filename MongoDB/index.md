# MongoDB
- MongoDB结构
  - 数据库
  - 集合
  - 文件

## 基本命令
| 命令                 |                                 描述 |
| :------------------- | -----------------------------------: |
| ``show dbs``         |                       显示已有数据库 |
| ``use admin``        | 进入数据，也可以理解成为使用数据库。 |
| ``show collections`` |                   显示数据库中的集合 |
| ``db``               |                         显示当前位置 |
